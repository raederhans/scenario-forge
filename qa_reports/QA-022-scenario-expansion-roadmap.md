## 二、推荐新剧本（按优先级排序）

### 第一优先级：Kaiserreich 1936

**可行性: 高** — 调色板已就位，构建管道可直接复用，mod文件已安装

| 维度 | 详情 |
|------|------|
| 场景 | 中欧帝国赢得一战的架空1936年 |
| 目标受众 | HOI4玩家（Kaiserreich是最受欢迎的mod，200万+订阅） |
| 调色板 | `kaiserreich`（已存在） |
| 构建方式 | `build_hoi4_scenario.py` 指向 Kaiserreich mod 目录 |
| 依赖 | Kaiserreich mod 游戏文件（已安装） |

**关键政治实体：**
- 德意志帝国 — 保留皇帝，主导欧洲，拥有大量海外殖民地
- 法国公社（Commune of France）— 工团主义革命后的法国
- 英国联合会（Union of Britain）— 社会主义革命后的英国
- 流亡法国/英国（National France / Canada）— 流亡政府
- 俄罗斯国 — 非苏联的俄罗斯（内战后重建）
- 奥匈帝国 — 可能以多民族联邦形式存在
- 奥斯曼帝国 — 衰落但未解体
- 德属非洲殖民地 — Mittelafrika 等

**实施步骤：**
1. 在 `init_map_data.py` 中添加 Kaiserreich scenario job
2. 运行 `build_hoi4_scenario.py`，指向 Kaiserreich mod 根目录
3. 审查自动生成结果，识别需要手动修正的区域
4. 创建 `data/scenario-rules/kaiserreich_1936.manual.json` 手动规则
5. 生成 `data/scenarios/kaiserreich_1936/` 全套数据文件
6. 验证 `kaiserreich.palette.json` 与国家标签匹配度
7. 注册到 `data/scenarios/index.json`
8. 运行 `check_scenario_contracts.py` 验证

### 第二优先级：1914 一战爆发

**可行性: 中** — 不依赖外部mod文件，纯手动规则构建，拓扑覆盖完美匹配

| 维度 | 详情 |
|------|------|
| 场景 | 1914年6月，一战爆发前夕的帝国时代欧洲 |
| 目标受众 | V3玩家（接近V3时间线末端）、历史爱好者 |
| 调色板 | `hoi4_vanilla`（复用现有） |
| 构建方式 | 基于现有拓扑 + 手动规则文件 |
| 依赖 | 无外部依赖 |

**关键政治实体与版图变化（相比1936）：**

| 实体 | 1914状态 | 1936对比 |
|------|----------|----------|
| 德意志帝国 | 含阿尔萨斯-洛林、波森、西普鲁士 | 1936失去这些区域 |
| 奥匈帝国 | 完整的二元帝国（奥地利+匈牙利+波黑+加利西亚+捷克+斯洛伐克+克罗地亚+特兰西瓦尼亚） | 1936已解体为7+个国家 |
| 奥斯曼帝国 | 含叙利亚/伊拉克/巴勒斯坦/汉志/也门（已失利比亚） | 1936仅剩土耳其 |
| 沙皇俄国 | 含芬兰、波兰、波罗的海三国、比萨拉比亚 | 1936苏联（失芬兰/波兰/波罗的海） |
| 大英帝国 | 含爱尔兰全岛 | 1936失去南爱尔兰 |
| 塞尔维亚 | 独立小国 | 1936已为南斯拉夫 |
| 黑山 | 独立小国 | 1936并入南斯拉夫 |
| 意大利王国 | 缺南蒂罗尔/伊斯特里亚/扎达尔 | 1936已获取这些 |
| 比利时 | 含刚果殖民地 | 同 |
| 希腊 | 较小版图（缺色雷斯/爱琴海岛屿部分） | 1936版图更大 |
| 罗马尼亚 | 小罗马尼亚（缺特兰西瓦尼亚/比萨拉比亚） | 1936大罗马尼亚 |
| 保加利亚 | 含西色雷斯/南多布罗加 | 1936失去这些 |

**不存在的国家（1914年）：**
- 波兰（被瓜分于俄/德/奥匈）
- 捷克斯洛伐克
- 南斯拉夫
- 芬兰
- 爱沙尼亚/拉脱维亚/立陶宛（沙俄领土）
- 爱尔兰（英国一部分）
- 匈牙利（奥匈内部）

**估算手动规则量：** ~30-40 条
**主要工作：** 将奥匈、奥斯曼、沙俄内部的feature重新分配到各帝国标签下

### 第三优先级：1945 二战结束

**可行性: 中** — 基于 hoi4_1939 增量规则，可能需要自定义拓扑表达占领区

| 维度 | 详情 |
|------|------|
| 场景 | 1945年5月，二战欧洲战场结束 |
| 目标受众 | HOI4玩家、冷战历史爱好者 |
| 调色板 | `hoi4_vanilla` |
| 构建方式 | hoi4_1939 + delta 规则 |

**关键版图变化（相比1939）：**
- 德国四区占领（美/英/法/苏）
- 奥地利四区占领
- 波兰东移（失克雷西，获西里西亚/波美拉尼亚/东普鲁士南部）
- 苏联扩张（吞并波罗的海三国、东普鲁士北部/柯尼斯堡、比萨拉比亚/外喀尔巴阡）
- 南斯拉夫联邦重建（含伊斯特里亚/扎达尔）
- 意大利失去伊斯特里亚、多德卡尼斯群岛
- 法国恢复阿尔萨斯-洛林

**特殊挑战：** 德国/奥地利占领区需要在现有拓扑的state级别上划分，可能需要自定义拓扑或利用sub-state级feature

**估算手动规则量：** ~25-35 条

### 第四优先级：1991 冷战结束

**可行性: 高（规则量最少）** — 接近现代版图，基于 modern_world 反向delta

| 维度 | 详情 |
|------|------|
| 场景 | 1991年12月，苏联正式解体 |
| 目标受众 | Cold War mod玩家、现代史爱好者 |
| 调色板 | `hoi4_vanilla` |
| 构建方式 | modern_world + delta 规则（合并一些现代独立国家） |

**关键版图差异（相比modern_world）：**
- 苏联仍存在（或刚解体，15个加盟共和国独立）
- 南斯拉夫开始解体（斯洛文尼亚/克罗地亚刚独立，波黑战争即将开始）
- 德国刚统一（1990）
- 捷克斯洛伐克仍为一个国家（1993才分裂）
- 厄立特里亚尚未独立（1993）
- 也门刚统一（1990）

**估算手动规则量：** ~15-20 条（最少）

## 三、资源可行性评估

### 3.1 拓扑覆盖 vs 剧本需求

| 剧本 | 拓扑够用？ | 说明 |
|------|-----------|------|
| Kaiserreich 1936 | **是** | HOI4 mod共享相同地图 |
| 1914 一战 | **是** | 欧洲+中东+北非完美覆盖 |
| 1945 二战结束 | **基本是** | 可能需要占领区细分 |
| 1991 冷战结束 | **是** | 接近现代版图 |
| 1918 凡尔赛 | **是** | 欧洲+中东覆盖足够 |
| Red Flood 1936 | **是** | HOI4 mod共享相同地图 |

### 3.2 构建管道复用

| 剧本 | 管道 | 额外工作 |
|------|------|----------|
| Kaiserreich 1936 | `build_hoi4_scenario.py` 直接复用 | 添加 scenario job + 手动规则 |
| Red Flood 1936 | `build_hoi4_scenario.py` 直接复用 | 添加 scenario job + 手动规则 |
| 历史剧本 (1914/1918/1945/1991) | 需要新的构建方式 | 手动规则 + 可能需要简化的构建脚本 |

### 3.3 长期全球拓扑扩展（方向性）

当前拓扑以欧洲为中心。未来如果要支持完整的 V3 1836 或 EU4 1444：
- **第一步：** 集成已有的 NA topology (v1/v2)
- **第二步：** 东亚/南亚拓扑数据（HOI4太平洋战争、V3中国/印度）
- **第三步：** 撒哈拉以南非洲（殖民地历史）
- **第四步：** 南美

这些是长期方向，不阻塞上述六个剧本的实施。

## 五、验证清单

每个新剧本完成后需要通过：

- [ ] `check_scenario_contracts.py` 数据完整性验证
- [ ] 浏览器加载测试：所有国家正确着色
- [ ] 边界一致性：与历史/架空设定对照
- [ ] audit.json feature coverage 无遗漏区域
- [ ] 调色板匹配：所有国家标签有对应颜色
- [ ] 剧本切换测试：从其他剧本切换到新剧本无报错
- [ ] 性能测试：加载时间在可接受范围内
