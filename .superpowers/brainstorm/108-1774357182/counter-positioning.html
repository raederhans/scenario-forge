兵牌定位：从"随机放置"到"智能布局"

当前问题：同一 feature 上多个兵牌的位置混乱

当前行为
目标：有序排列

布局策略

A

Centroid 网格布局

以 feature 的 centroid 为中心，按网格排列同一 feature 内的所有兵牌。1个=正中心，2个=左右排列，3个=三角，4个=2×2 方阵。超过4个时使用堆叠+徽章（已有功能）。

适合：feature 较大的区域，兵牌数量适中

B

始终堆叠 + 点击展开

同一 feature 上的兵牌始终堆叠为一组（最多3层可见卡片），点击堆叠后弹出展开面板显示所有兵牌。这保持地图整洁，信息通过交互获取。

适合：兵牌密集的区域，强调地图可读性

C

混合：自动布局 + 堆叠阈值 (推荐)

设一个阈值（如3个）：≤3个时用 centroid 网格排列，>3个时自动堆叠+徽章。用户仍可手动拖拽调整位置。布局在 zoom 变化时自动调整——远距离全部堆叠，近距离展开排列。

兼顾可读性和信息量，响应缩放级别
