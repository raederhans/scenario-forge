# 导出回归手测清单

> 每次导出链路变更后，至少完整执行一次。

## A. 图层一致性

- [ ] 图层顺序一致性：导出图像中的上下覆盖关系与预览一致。
- [ ] 图层显隐一致性：关闭图层后导出不应出现该图层内容；重新开启后恢复。
- [ ] 文本层导出一致性：文本位置、字号、旋转与预览一致，无丢字、错位、锯齿异常。
- [ ] 预览真实性：`Main image / Single layer` 都显示真实预览，不再出现 placeholder 文案。
- [ ] Text stack 一致性：`Render-pass labels` 与 `SVG annotations` 的开关会同时影响预览和导出。

## B. 缓存与重复导出

- [ ] 烘焙后再次导出命中缓存行为符合预期（无异常重建、无明显回退）。
- [ ] 连续重复导出同配置时，性能与结果稳定。

## C. 快速导出回退

- [ ] `composite` 路径可稳定完成单文件下载。
- [ ] `per-layer` 路径可稳定完成多文件下载。
- [ ] `bake-pack` 路径可稳定完成 baked PNG + metadata JSON 多文件下载。

## D. 失败分级提示

- [ ] 参数错误：出现参数错误级别提示，并有明确恢复操作。
- [ ] 内存不足：出现内存不足级别提示，并有降分辨率建议。
- [ ] 跨域 SVG 失败：出现跨域 SVG 级别提示，并有同源替换建议。

## E. 约束策略

- [ ] 8K 上限生效：超过 7680×4320 的导出请求被拒绝。
- [ ] 单次并发导出限制生效：同一时间不超过约定并发数（当前为 1）。
- [ ] 预览与导出分辨率分离生效：切换导出倍率不影响编辑预览分辨率。
- [ ] Project → Export 区块可独立打开 overlay workbench，不再依赖 Utilities。
- [ ] `view=export` URL restore 会打开 Export 一级区块并拉起 overlay。
