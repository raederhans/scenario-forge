# 导出回归手测清单

> 每次导出链路变更后，至少完整执行一次。

## A. 图层一致性

- [ ] 图层顺序一致性：导出图像中的上下覆盖关系与预览一致。
- [ ] 图层显隐一致性：关闭图层后导出不应出现该图层内容；重新开启后恢复。
- [ ] 文本层导出一致性：文本位置、字号、旋转与预览一致，无丢字、错位、锯齿异常。

## B. 缓存与重复导出

- [ ] 烘焙后再次导出命中缓存行为符合预期（无异常重建、无明显回退）。
- [ ] 连续重复导出同配置时，性能与结果稳定。

## C. 快速导出回退

- [ ] 快速导出路径可用（常见参数下可稳定完成下载）。
- [ ] 主导出失败时，快速导出回退可触发并成功生成文件（如项目启用该策略）。

## D. 失败分级提示

- [ ] 参数错误：出现参数错误级别提示，并有明确恢复操作。
- [ ] 内存不足：出现内存不足级别提示，并有降分辨率建议。
- [ ] 跨域 SVG 失败：出现跨域 SVG 级别提示，并有同源替换建议。

## E. 约束策略

- [ ] 8K 上限生效：超过 7680×4320 的导出请求被拒绝。
- [ ] 单次并发导出限制生效：同一时间不超过约定并发数（当前为 1）。
- [ ] 预览与导出分辨率分离生效：切换导出倍率不影响编辑预览分辨率。
