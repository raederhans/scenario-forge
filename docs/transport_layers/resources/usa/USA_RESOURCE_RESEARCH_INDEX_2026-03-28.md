# 美国工业矿产、能源设施、工业区研究总览

日期：2026-03-28

## 一句话结论

美国这三条线里，`工业矿产` 与 `发电设施` 都能用强官方源先落一版，但 `工业区` 没有日本那样现成的联邦统一主层，首版必须接受“官方缺口明确存在”这个事实。

## 三条线总体成熟度判断

| 线别 | 总体成熟度 | 当前最强来源判断 | 几何 | 是否存在全国统一官方主源 | 与日本相比最明显的不同 |
|---|---|---|---|---|---|
| 工业矿产 | 高 | 以 USGS 官方组合源为主：MRDS 负责全国广覆盖点位骨架，Alaska ARDF 负责阿拉斯加，USMIN 负责重点矿床和关键矿种增强 | 以点为主，少量面/区带需后处理成代表点 | 没有像日本 GSJ 那样单一而整洁的一张全国主层，但组合后可用 | 日本更像“一张主表”；美国更像“同一官方体系内的组合主源” |
| 能源设施 | 中高 | 发电设施非常强，EIA-860 已足够做全国点层；炼厂、LNG 等有官方子类源，但没有一个联邦点层把“全部能源设施”一次收口 | 点 | 对“发电设施”有；对“全部能源设施”没有 | 日本主源较旧但概念集中在发电设施；美国发电数据更新更强，但大类边界更容易溢出 |
| 工业区 | 低 | 没有可接受的联邦统一主源；联邦公开源多是土地覆被、就业密度、规划/统计代理，不是“真实工业园区/工业用地” | 面 | 没有 | 日本至少有 MLIT L05 这种贴题的真实工业用地层；美国这一条线缺口明显更大 |

## 哪条线最适合先试点

如果产品名称必须保持现在这三条线不改，最适合先试点的是 `工业矿产`。

原因很简单：

- 它的产品语义就是“矿床/资源分布”，不要求经营状态，也不要求网络连通。
- 虽然美国没有日本那样一张单表，但 USGS 官方组合源已经足够搭出全国骨架。
- 它和现有仓库的点图层承接最直接，风险比把 `energy_facilities` 先做成“实质上主要是 power plants”更小。

如果后续允许把 `energy_facilities` 首版明确命名成 `发电设施为主`，那能源线会是第二优先，工程难度甚至可能更低。

## 和日本最明显的差异

最明显的差异不是“美国数据更多”，而是“美国更依赖按主管机构拆分的数据治理”。

- 日本在这三条线上，尤其工业用地，能找到更贴题的全国专题层。
- 美国在矿产和能源上更强，是因为 USGS、EIA 这类专业机构长期维护专题数据。
- 美国在工业区上更弱，是因为真实工业园区/工业用地的管理更分散，联邦层更常提供代理指标，而不是直接提供园区面。

## 哪些地方必须降级到非官方但可信公开源

必须明确降级的只有 `工业区` 主体几何。

- 现实可行的全国统一开放面源，当前只能降级到 Tier C 的 [OpenStreetMap / Geofabrik](https://download.geofabrik.de/north-america/us.html) `landuse=industrial`、`industrial=*`、`industrial_park` 等对象。
- 但这只能当“全国统一的 provisional geometry”，不能当官方事实本体。
- 如果后续要把名字、边界、管理主体做实，仍然要回到州、市、港区或地方开发机构公开资料逐块核对。

能源线里也有一个较小的降级点：

- 如果一定要在首版就纳入炼厂、LNG 接收/出口终端、气体处理厂等多种非发电设施，官方源是分散的。
- 这时可以先用 EIA / FERC / DOE 官方目录做主，必要时再用 Tier C 行业目录做核点，但不应把 Tier C 反过来抬成全国主源。

## 与现有仓库架构的承接判断

- `mineral_resources`：继续按点层走，复用现有点位图层承接链路；USMIN 的面/区带记录在入库时转代表点。
- `energy_facilities`：首版如果以发电设施为主，也应按点层走；不要一开始就承诺“全能源设施统一主层”。
- `industrial_zones`：必须按 polygon/context layer 走，不能点化；否则会把“真实园区/工业用地”的语义直接做坏。

## 风险与下一步建议

1. 不要把“有全国源”误写成“有完美全国源”。美国矿产和能源都更像强骨架，不是零缺口全闭合。
2. 工业区要尽早接受“联邦主源缺失”的结论，不要用 NLCD、EPA Smart Location Database 或各类 zoning 代理偷偷顶上。
3. 第一轮落地建议顺序：
   1. `mineral_resources`
   2. `energy_facilities（首版明确以发电设施为主）`
   3. `industrial_zones（先做缺口研究和试点城市验证，不先承诺全国统一）`
4. 和日本不同，美国首版更需要在产品文案里写清楚边界，否则用户会自然把“energy facilities”理解成已经覆盖炼厂、LNG、油库、变电站等全部能源节点。
